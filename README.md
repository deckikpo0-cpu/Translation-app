# Speak & Translate Notebook

🎤 A browser-based **Speak & Translate Notebook** with interactive history, multilingual speech, and live translation.

## Features

- 🎙️ **Live speech recognition**: Speak in English and see it transcribed.
- 🌐 **Translate to 20+ languages** including:
  - Common: Spanish, French, German, Chinese, Japanese, Korean, Arabic, Hindi
  - Nigerian: Igbo, Yoruba, Hausa, Nigerian Pidgin
- 🗣️ **Multilingual Text-to-Speech**: Plays translation in correct language if supported by your browser.
- 🕘 **Interactive history**: Replay original or translated text.
- ✏️ **Editable translations**: Click history entries to edit and re-translate.
- 🔢 **Numbers to words** conversion for natural pronunciation.
- 🎨 **Animated translations**: Display letters one by one for a smooth effect.
- ✅ **Completely free** using [MyMemory API](https://mymemory.translated.net/).

## How to Use

1. Open `index.html` in your browser (Chrome, Edge, or Firefox recommended).
2. Select a target language from the dropdown.
3. Click **Start Speaking** and speak in English.
4. The app translates and animates the text in the target language.
5. Click **Speak Translation** to hear it.
6. Click **Re-Translate Edited Text** to translate modified text.
7. Use the **history section** to replay or edit previous translations.

## Supported Browsers

- Chrome, Edge, Firefox (latest versions recommended for SpeechRecognition and SpeechSynthesis support).
